I host a non insignificant amount of things for myself. These are spread on a vps and a home server.

My home server is an old Acer Aspire TC-220 prebuilt with the these specs.

CPU : AMD A10-7800 12-core GPU : AMD Radeon R7 360 RAM : 12GB of missmatched DDR3 Storage :

OS drive : Kingston ssdNOW 128GB SSD (RBU-SC150S37128GD)

Data drive : Western Digital AV-25 320GB 2.5" 5400RPM (WD3200BUCT-63TWBY0) (has now died)

OS : proxmox

Its not powerful or fancy but it works.

List of things I host on my home server :

AdGuard Home(network wide adblocker and DNS rewriter)
Caddy(SSL reverse proxy)
Homeassistant(Smarthome)
Remotely(Remote access)
Uptime Kuma(Monitoring tool)
Planka(Project tracker)

On the server I got given for free I run

FreshRSS(RSS feed aggregator)
docker-mailserver(An email server)
Opengist(Selfhosted git powered pastebin)
Caddy(SSL reverse proxy and webserver for this site)
