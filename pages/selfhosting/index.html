I host a non insignificant amount of things for myself. These are spread on 2 vps' and a home server.

My home server is an old Acer Aspire TC-220 prebuilt with the these specs

CPU : AMD A10-7800 12-core GPU : AMD Radeon R7 360 RAM : 12GB of missmatched DDR3 Storage :

OS drive : Kingston ssdNOW 128GB SSD (RBU-SC150S37128GD)

Data drive : Western Digital AV-25 320GB 2.5" 5400RPM (WD3200BUCT-63TWBY0)

OS : openmediavault

Its not powerful or fancy but it works.

List of things I host on my home server :

AdGuard Home(network wide adblocker and DNS rewriter)
Authentik(Oauth provider)
Caddy(SSL reverse proxy)
Crafty Controller(Minecraft server manager)
FreshRSS(RSS feed aggregator)
Homeassistant(Smarthome)
Jellyfin(Media server)
Kanboard(A kanban manager)
Linkding(A link manager)
n.eko(Virtual browser)
Openspeedtest(Selfhosted speedtest)
Privatebin(Private pastebin/secret sharing)
Remotely(Remote access)
Stump(Media server for books)
Uptime Kuma(Monitoring tool)
WireGuard Easy(VPN server) \

One of the VPS' I have is from a commercial hoster and one has been given to me for free.

The commerial one is a 2TB Big storage server from crunchbits.
